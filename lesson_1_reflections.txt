 # documentos com reflexões sobre comandos do git

* git diff file_1 file_2: permite descobrir quais as modificações introduzidas 
entre duas versões.

* git log: show the log of modifications;
* git log --stat: show the log of modifications + the number of modificatios;

* git checkout: permite tornar uma versão especifica a versão atual e poder 
execut-la;

* git commit: 1 sempre q houver uma mudança logica (texto, codigo), durante deferentes etapas
